# VPN IKEv2 + скрипт для IOS
### Простая и быстрая установка VPN соединения
___

#### Требования:
+ Сервер Ubuntu 20.04
___
## Шаг 1.
Качаем проект:
```sh
git clone https://github.com/lastlibertarian/IKEv2_for_ios /root/IKEv2_for_ios
```
Запускаем скрипт:
```sh
python3 /root/IKEv2_for_ios/main.py
```
